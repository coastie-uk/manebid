Deployment Guides

Choose how to install ManeBid

Both deployment forms are maintained in the same repository and run the same application. Choose the approach that best fits your server.

Installation

Installation (bare)

Install Node.js directly and run ManeBid with systemd or PM2, using Apache and Let's Encrypt for the public frontend and HTTPS.

Open bare installation guide

Installation (Docker)

Run the backend and Caddy frontend with Docker Compose, runtime secret mounts, persistent volumes, and automatic HTTPS.

Open Docker installation guide
